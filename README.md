



# Netflix Data Analysis Using Python

## Project Overview

This project analyzes Netflix movies and TV shows to identify trends and patterns in the platform's content library. The analysis uses Python and data visualization techniques to explore content types, release years, ratings, genres, countries, and movie durations.

The goal of this project is to demonstrate practical skills in **data cleaning, exploratory data analysis (EDA), data visualization, and insight generation**.

---

## Objectives

* Analyze the distribution of Movies and TV Shows on Netflix.
* Identify trends in Netflix content releases over the years.
* Find the countries contributing the most content.
* Analyze the most common content ratings.
* Explore popular genres and categories.
* Analyze movie duration patterns.
* Identify key trends and insights from the dataset.

---

## Technologies Used

* **Python**
* **Pandas** – Data manipulation and analysis
* **NumPy** – Numerical operations
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical data visualization
* **Jupyter Notebook** – Development environment

##  Dataset

The dataset contains information about Netflix movies and TV shows, including:

* Show ID

* Title
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genres
* Description

##  Data Analysis Process

### 1. Data Loading

The dataset was loaded into Python using Pandas.

### 2. Data Cleaning

* Checked for missing values.
* Checked for duplicate records.
* Handled missing or unavailable data.
* Prepared columns for analysis.

### 3. Exploratory Data Analysis (EDA)

The dataset was analyzed to identify trends and patterns across different categories.

### 4. Data Visualization

Charts and graphs were created using Matplotlib and Seaborn to present the findings clearly.
